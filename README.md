# Asteroid Stopwatch
A stopwatch app for [AsteroidOS](http://asteroidos.org/). This stopwatch app has been
modified from the original work to make it prevent falling to sleep.

## Dependencies
* [qml-asteroid](https://github.com/AsteroidOS/qml-asteroid)
